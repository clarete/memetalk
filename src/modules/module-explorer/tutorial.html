Sometimes we want to explore complex objects, its properties and methods. Memetalk offers a Inspector window -- through Inspect It action -- that allows us to see the structure of an object, change its properties and send messages to it. Try it, selecting the following code and typing CTRL+I.

A new instance of the class Workspace should be created with the "new" message and a Inspector window should appear. The Inspector shows the properties of that instance on the left pane, and their values on the right (currently, the cryptic strings appearing as property values are python string representations for memetalk objects. In the future, a more readable representation for them should appear there). Double clicking on a property name opens another Inspector for the object bound to that property. Messages can be sent to the Workspace instance on the line edit at the bottom of the window. The instance is bound to "this" keyword. Try typing "this.show()" on that field and hitting Enter. The Workspace window should appear as response to that message.

A Workspace is an editor with all the features discussed above, useful for exploring and executing code.
